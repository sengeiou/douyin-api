## 1、弹幕如何采集？
​

### 请求Api
```http
http://主机地址/douyin/liveroom/chat?token=xxx&room_id=6843198199583378191
```

### 

### 请求方式
```http
GET
```

### 

### 参数
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| token | string | 接口授权码 |
| room_id | int | 直播间id |


>**短视频直播数据采集接口SDK请**[点击查看接口文档](https://docs.qq.com/doc/DU3RKUFVFdVhQbXlR)
