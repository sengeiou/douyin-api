# 抖音商城精选商品列表接口数据采集

## 抖音Api：带货商品详情

### 请求Api
```http
http://主机地址/douyin/commerce/feed?token=xxx&tab_id=1

```

### 

### 请求方式
```http
GET
```

### 

### 参数
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| token | string | 接口授权码 |
| tab_id	 | int | 商品分类id，详见“商城商品分类” |
| page_num	 | int | 翻页，默认1，翻页：1、2、3、4…… |
