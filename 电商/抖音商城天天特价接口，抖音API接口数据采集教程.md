# 抖音商城天天特价接口，抖音API接口数据采集教程



## 抖音商城新品接口

### 请求Api
```http
http://主机地址/douyin/commerce/seckill_page?token=xxx

```


### 请求方式
```http
GET
```

### 

### 参数
| 字段 | 类型 | 说明 |
| --- | --- | --- |
| token | string | 接口授权码 |
| cursor | int | 翻页游标，初始为1，第二页则传2，以此类推 |
